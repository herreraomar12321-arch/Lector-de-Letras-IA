Requisitos previos: Tener Python instalado en el equipo.

1-Abrir la terminal: Vaya a la carpeta Red Neuronal Convolucional, la cual contiene el index.html y la carpeta tfjs_modelo. Hacer clic derecho en un espacio vacío y seleccione "Abrir en Terminal" (o PowerShell/CMD).

2-Iniciar el servidor: Escriba el siguiente comando y pulse Enter: python -m http.server 8000 (Debe aparecer un mensaje diciendo "Serving HTTP on :: port 8000") o similar.

3-Abrir en el navegador: Abra su navegador (Chrome/Edge recomendado) y entrar a: http://localhost:8000/

4-Probar el modelo:

	Dibuje una letra en el recuadro negro (Dibuje la letra GRANDE y centrada para mejor precisión).

	Pulse "PREDECIR".
